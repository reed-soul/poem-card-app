# 🏮 诗词日历

每日一首诗词，与天地对话。

## 特性

- ✨ 每日推荐一首精选古诗词
- 🌤️ 根据天气、节气智能推荐
- 🎨 精美的中国风卡片设计
- 💚 跨平台支持（Windows & macOS）
- 📚 离线可用，本地数据库

## 技术栈

- **框架**: Electron + React + TypeScript
- **样式**: Tailwind CSS
- **动画**: Framer Motion
- **数据**: 本地SQLite + 智谱AI

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```
